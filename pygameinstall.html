Installieren des Pakets pygame in Thonny

Möglichkeit 1 — am einfachsten

Öffne Thonny, kopiere den folgenden Code und führe diesen einmal aus: 


import sys
import subprocess
try:
    import pygame
except ImportError:
    print("pygame is not installed. Installing now...")
    subprocess.check_call([sys.executable, '-m', 'pip', 'install', 'pygame'])
    import pygame
Gratulation, du hast pygame installiert und kannst es fortan verwenden.

Möglichkeit 2 — hat 1. nicht funktioniert oder weshalb willst du das so machen?

Thonny installiert das Paket (Bibliothek) pygame nicht automatisch. Darum müssen wir das Paket selbst nachladen. Dazugehst du wie folgt vor:

  1. Öffne Thonny und dort die Paketverwaltung.
  2. Gib im Suchfenster (1.) "pygame" ein, klicke auf Suchen (2.) und wähle dann bei den Ergebnissen pygame (3.).
  3. Klicke auf installieren. Danach kannst du in Thonny pygame verwenden.
