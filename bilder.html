Bilder anzeigen

Ein Bild muss zuerst als Element erstellt und die Bilddaten geladen werden (Dokumentation image.load()). Wenn das Bild im selben Ordner wie das Programm liegt, dann reicht es den Dateinamen anzugeben. Enthält das Bild Transparenz (z.B. um den Hintergrund unsichtbar zu machen), so muss zusätzlich noch der Befehl convert_alpha() aufgerufen werden. 


bild = pygame.image.load("mein_bild.jpg") # ohne Transparenz
bild_transparent = pygame.image.load("mein_bild_mit_transparenz.png").convert_alpha() # mit Transparenz
Hat man das Bild-Element erstellt, kann man das Bild beliebig oft mit dem Befehl blit() zeichnen. Neben dem Bild müssen dort noch die Koordinaten links oben angegeben werden. 

FENSTER.blit(bild, (100, 150))
