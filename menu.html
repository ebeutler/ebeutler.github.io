Menü (oder verschiedene Ansichten) erstellen

Grundsäztlich wird die Anzeige ja viele Male pro Sekunde neu gezeichnet. Will man also verschiedene Ansichten haben - z.B. ein Menü und ein Spielbildschirm - so muss irgendwie entschieden werden was jeweils neu gezeichnet werden soll. Am einfachsten geht das mit einer Status-Variablen in der man speichert ob man sich aktuell gerade im Menü befindet oder nicht. Das könnte in etwa so aussehen:


if show_menu:
    # Zeige das Menü an

    # Wird auf eine bestimmte Stelle geklickt oder eine Taste gedrückt kann mit
    # show_menu = False
    # umgestellt werden und der Spielbildschirm wird angezeigt.
else:
    # Zeige den Spielbildschirm an
