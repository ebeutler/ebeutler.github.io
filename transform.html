Verformungen

Zur Vollständigen Dokumentation über Verformungen: transform

 

Elementen (Surfaces) können in pygame auf viele verschiedene Weisen verformt werden. Hier die üblichsten, weitere sind in der offiziellen Dokumentation zu finden. In den Beispielen wird jeweils davon ausgegangen, dass ein, in der Variable bild gespeichertes, Bild verformt werden soll.

  • Rotation (drehen)

    Mit dem Befehl rotate() kann ein Element gedreht werden. Neben dem zu drehenden Element muss man ausserdem die Grad angeben um die gedreht werden soll. Im Beispiel wird das Bild um 45° nach rechts gedreht.

    
rotiert_45grad_rechts = pygame.transform.rotate(bild, -45)
    Etwas komplizierter ist es nicht ein Bild, sondern z.B. ein Rechteck, ein Kreis oder eine Ellipse, zu drehen. Um das zu erreichen muss man die Form auf neue kleine Zeichenfläche zeichnen. Man kann sich das so vorstellen, wie wenn man beim Zeichnen auf Papier die Ellipse nicht direkt aufs Zeichenblatt zeichnet sondern auf ein Post-It und danach das Post-It dreht und in der gewünschten Drehung auf das eigentliche Blatt klebt. Mit pygame sieht das dann wie folgt aus, wenn man eine 160x80 Pixe grosse ellipse zeichen und um 45 Grad drehen will. 
    
# Zeichnen einer Ellipse auf eine transparente kleine Zeichenfläche
zeichen_fläche = pygame.Surface((160, 80), pygame.SRCALPHA, 32).convert_alpha()
ellipsen_fläche = pygame.Rect((0,0), (160, 80))
pygame.draw.ellipse(zeichen_fläche, "#FFFFFF", ellipsen_fläche)

# rotieren der Ellipse um 45° und Zeichnen der rotierten Ellipse
rotierte_fläche = pygame.transform.rotate(ohr, -45)
FENSTER.blit(rotierte_fläche, (300, 200))

  • Skalieren (verkleinern & vergrössern)

    Zum verkleinern oder vergrössern von Elementen gibt es zwei Möglichkeiten. Ändern der Grösse auf fixe Dimensionen und proportionale Änderungen. Mit dem Befehl scale() kann ein Element auf eine feste Grösse geändert werden. Im Beispiel wird das Bild auf die Dimensionen 100 x 50 geändert. Dabei kann das Element "gequetscht" (oder gedehnt) werden, wenn man die Seitenverhältnisse ändert. Mit dem Befehl scale_by() kann, bei gleichbleibenden Seitenverhältnissen, verkleinert oder vergrössert werden. Zahlen <1 verkleinern, Zahlen >1 vergrössern das Element.
    Da, speziell beim Vergrössern, ein Bild "verpixeln" kann gibt es auch noch die Befehle smoothscale() (bzw. smoothscale_by()), welche die Kanten etwas schöner machen, dafür aber etwas langsamer sind. Diese zwei Befehle sind gleich zu verwenden wie die zwei oben beschriebenen.

    
skaliert_100x50 = pygame.transform.scale(bild, (100, 50))
skaliert_halb = pygame.transform.scale_by(bild, 0.5)

  • Spiegeln

    Ein Element kann mit flip() horizontal und/oder vertikal gespiegelt werden. Neben dem Element muss man dabei angeben ob horizontal gespiegelt werden soll (True/False) sowie als zweites ob vertikal gespiegelt werden soll (True/False).

    
gespiegelt_horizontal = pygame.transform.flip(bild, True, False)
gespiegelt_vertikal = pygame.transform.flip(bild, False, True)
