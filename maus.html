Maussteuerung

Zur Vollständigen Dokumentation über die Maussteuerung: mouse

 

Die Mausposition kannst du mit dem Befehl get_pos() auslesen. Die x- und y-Koordinaten sind dabei relativ zur linken oberen Ecke des Programmfensters.


maus_pos = pygame.mouse.get_pos() # Mausposition als (x, y) auslesen
x = maus_pos[0]
y = maus_pos[1]

 

Um auf Klicks zu reagieren, musst du wieder die Events (Ereignisse) verwenden, wie schon im Grundaufbau beim schliessen des Fensters. Die Maus kann dabei die Ereignisse MOUSEBUTTONDOWN, MOUSEBUTTONUP, MOUSEWHEEL und MOUSEMOTION auslösen. Im Beispiel unten wird MOUSEBUTTONDOWN verwendet um einen Klick auf die Maus zu erkennen. Innerhalb der for-Schleife, die über alle Ereignisse iteriert kannst du dann in einem zusäztlichen elif prüfen ob es sich um so ein MOUSEBUTTONDOWN-Ereignis handelt. Wenn ja kannst du prüfen ob Taste 0 (links), 1 (mitte) oder 2 (rechts) geklickt wurde. (Offizielle Dokumentation: get_pressed()) 


for event ...
  if event.type ...
    .....
  elif event.type == pygame.MOUSEBUTTONDOWN:
    if pygame.mouse.get_pressed()[0]: # Links-Klick
      # Mach was wenn links geklickt wurde
    elif pygame.mouse.get_pressed()[2]: # Rechts-Klick
      # Mach was wenn rechts geklickt wurde
