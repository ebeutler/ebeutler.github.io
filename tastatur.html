Tastatureingaben

Zur Vollständigen Dokumentation über die Tastatursteuerung: key

 

Um auf Tastatureingaben zu reagieren, musst du wieder die Events (Ereignisse) verwenden, wie schon im Grundaufbau beim schliessen des Fensters. Die Tastatur kann dabei die Ereignisse KEYDOWN und KEYUP auslösen. Jeder Tastendruck löst zuerst KEYDOWN aus und beim Loslassen der Taste wieder ein KEYUP. Im Beispiel unten führen beide Ereignisse zu einem Ausführen des Codes. Innerhalb der for-Schleife, die über alle Ereignisse iteriert kannst du in einem zusäztlichen elif prüfen ob es sich um so ein Tastatur-Ereignis handelt. Wenn ja, dann hat das Ereignis zwei Variablen: key (welche Taste gedrückt wurde) und mod (welche Modifikatoren - z.B. ALT, SHIFT, ... - gedrückt wurden). Damit kannst du prüfen was genau gedrückt wurde und entsprechend darauf reagieren. Eine vollständige Liste aller Tasten-Kürzel (die du im if zum Vergleichen benötigst) findest du in der vollständigen Dokumentation (key).


for event ...
  if event.type ...
    .....
  elif event.type == KEYDOWN or event.type == KEYUP:
    if event.key == K_a and (event.mod & pygame.KMOD_SHIFT):
        # Mach was wenn 'A' gedrückt
        print('GROSSES A')
    elif event.key == K_a:
        # Mach was anderes wenn 'a' gedrückt
        print('kleines a')

 

Alternativ kann man mit dem Befehl get_pressed() auch jederzeit abfragen welche Taste(n) gerade gedrückt wird/werden.


taste = pygame.key.get_pressed()
if taste[K_a]:
  # a wurde gedrückt
